# 📌 Project Overview

This repository contains a **Manual Testing Project** performed on the **Spotify Music Streaming Application**.

The goal of this project is to demonstrate **Software Testing Life Cycle (STLC)** practices including test planning, test case design, bug reporting, and test execution.

---

# 🎯 Objectives

* Verify core functionalities of the Spotify application
* Identify defects in the application
* Validate application usability and performance
* Demonstrate manual testing documentation

---

# 🧪 Modules Tested

The following modules were tested during the testing process:

1. User Login / Authentication
2. Music Search
3. Song Playback
4. Playlist Management
5. Volume Control
6. Logout Functionality

---

* Functional Testing

* Regression Testing
